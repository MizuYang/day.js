- day.js 日期套件
  - 多國語系
    - 翻譯成各國語系的月份、週 名稱
  - 日期比較大、小
    - 可自行輸入日期
  - 日期加減
    - 今天的日期
      - 可自行設定增加、減少
      - 可自行更改數值
      - 可自行更改單位
      - 上方會有計算日期 加、減 的公式
    - 指定的日期
      - 可自行設定日期
      - 可自行設定增加、減少
      - 可自行更改數值
      - 可自行更改單位
      - 上方會有計算日期 加、減 的公式
